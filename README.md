## Convenção de commits

```
feat(área): adiciona novo conteúdo
docs(área): melhora documentação ou notas
refactor(área): reorganiza ou melhora código existente
fix(área): corrige erro em query ou script
exercise(área): adiciona ou resolve exercício
```
